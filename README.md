# Smart Notes Manager

## Problem
Students struggle to organize study notes efficiently.

## Solution
A Java-based console application to manage notes with
categorization, searching, and persistent storage.

## Features
- Add notes
- View notes
- Search notes
- Delete notes
- File storage

## Technologies
Java, OOP, File Handling

## How to Run
1. Clone repository
2. Open in IntelliJ/Eclipse
3. Run Main.java
